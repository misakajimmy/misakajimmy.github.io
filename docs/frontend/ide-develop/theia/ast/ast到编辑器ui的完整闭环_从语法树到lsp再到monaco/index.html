InversifyJSпјҡжҮ’жіЁе…ҘдёҺеҫӘзҺҜдҫқиө–пјҲlazyInject зҡ„еҲ©дёҺејҠпјү

  • е…ҲеӣһйЎҫдёҖдёӢпјҡеҫӘзҺҜдҫқиө–еңЁ DI е®№еҷЁйҮҢдёәд»Җд№Ҳйә»зғҰпјҹ

  • жҮ’жіЁе…ҘдёҚжҳҜй“¶еј№пјҡе®ғи§ЈеҶізҡ„жҳҜвҖңи®ҝй—®ж—¶жңәвҖқпјҢдёҚжҳҜвҖңи®ҫи®Ўй—®йўҳвҖқ

  • е®№еҷЁеңЁжөӢиҜ•йҮҢзҡ„и§’иүІпјҡжөӢиҜ•еӨ№е…·

VS Code жҸ’д»¶дҪ“зі»жҖ»и§ҲпјҡExtension HostгҖҒжҝҖжҙ»дәӢд»¶дёҺиҙЎзҢ®зӮ№

  • жҸ’д»¶зҡ„еҹәжң¬з»“жһ„пјҡpackage.json + жҝҖжҙ»е…ҘеҸЈ

    • 2. жү©еұ•д»Јз ҒйҮҢзҡ„ activate / deactivate
  • жҝҖжҙ»дәӢд»¶пјҡжҸ’д»¶дёҚжҳҜдёҖдёҠжқҘе°ұе…Ёи·‘иө·жқҘзҡ„

  • DI е®№еҷЁе’Ңжү©еұ•зӮ№пјҡTheia еҰӮдҪ•з»„з»ҮжҸ’д»¶иғҪеҠӣпјҹ
  • еүҚз«Ҝжү©еұ•пјҡе‘Ҫд»Ө / иҸңеҚ• / еҝ«жҚ·й”® / и§ҶеӣҫжҳҜжҖҺд№ҲжҢӮиҝӣеЈійҮҢзҡ„пјҹ

  • е°Ҹз»“пјҡзҗҶи§Ј Theia жҸ’д»¶дҪ“зі»иҰҒжҠ“дҪҸзҡ„еҮ дёӘзӮ№

  • жһ¶жһ„еҲҶеұӮеҜ№жҜ”пјҡExtension Host vs еүҚз«Ҝ / еҗҺз«Ҝжү©еұ•
  • жү©еұ•зӮ№еЈ°жҳҺж–№ејҸпјҡcontributes vs жү©еұ•зӮ№жҺҘеҸЈ + DI з»‘е®ҡ
    • VS Codeпјҡpackage.json + contributes

  • е‘Ҫд»Ө / иҸңеҚ• / еҝ«жҚ·й”®пјҡдёӨиҫ№зҡ„еҜ№з…§е…ізі»
  • иҮӘе®ҡд№үи§ҶеӣҫдёҺзј–иҫ‘еҷЁеўһејәпјҡWebview / TreeView vs Lumino Widget / Monaco йӣҶжҲҗ
  • е…је®№жҖ§дёҺеӨҚз”ЁпјҡеңЁ Theia дёӯи·‘ VS Code жҸ’д»¶ж„Ҹе‘ізқҖд»Җд№Ҳпјҹ

  • е°Ҹз»“пјҡдёҖеј и„‘жө·йҮҢзҡ„еҜ№жҜ”еӣҫ

VS Code жҸ’д»¶пјҡе‘Ҫд»ӨгҖҒиҸңеҚ•дёҺеҝ«жҚ·й”®вҖ”вҖ”д»Һ contributes еҲ°д»Јз ҒжіЁеҶҢ

  • е‘Ҫд»ӨжҳҜд»Җд№Ҳпјҹе’ҢиҸңеҚ•гҖҒеҝ«жҚ·й”®жҳҜд»Җд№Ҳе…ізі»пјҹ
  • еңЁ package.json йҮҢеЈ°жҳҺе‘Ҫд»Өпјҡcontributes.commands
  • еңЁд»Јз ҒйҮҢжіЁеҶҢе‘Ҫд»Өе®һзҺ°пјҡvscode.commands.registerCommand
  • иҸңеҚ•пјҡе‘Ҫд»ӨеҰӮдҪ•еҮәзҺ°еңЁеҸій”®иҸңеҚ• / ж Үйўҳж ҸзӯүдҪҚзҪ®пјҹ
  • еҝ«жҚ·й”®пјҡдёәе‘Ҫд»Өз»‘е®ҡжҢүй”®з»„еҗҲ
  • дёҖжқЎе®Ңж•ҙй“ҫи·ҜеӨҚзӣҳпјҡд»Һ contributes еҲ°жңҖз»Ҳи§ҰеҸ‘
  • еёёи§Ғеқ‘дёҺе®һи·өе»әи®®

Theia жү©еұ•пјҡе‘Ҫд»ӨгҖҒиҸңеҚ•дёҺеҝ«жҚ·й”®вҖ”вҖ”еҹәдәҺ Command дёҺ Keybinding жү©еұ•зӮ№

  • Theia йҮҢзҡ„е‘Ҫд»Өзі»з»ҹиҙҹиҙЈд»Җд№Ҳпјҹ
  • CommandContributionпјҡжіЁеҶҢе‘Ҫд»ӨдёҺиЎҢдёә
  • еүҚз«ҜжЁЎеқ—дёӯзҡ„з»‘е®ҡпјҡжҠҠиҙЎзҢ®зұ»жҺҘе…Ҙ DI е®№еҷЁ
  • MenuContributionпјҡжҠҠе‘Ҫд»ӨжҢӮеҲ°иҸңеҚ•дёҠ
  • KeybindingContributionпјҡдёәе‘Ҫд»Өз»‘е®ҡеҝ«жҚ·й”®
  • дёҖжқЎе®Ңж•ҙй“ҫи·ҜпјҡTheia дёӯзҡ„е‘Ҫд»Ө / иҸңеҚ• / еҝ«жҚ·й”®жҳҜеҰӮдҪ•еҚҸеҗҢзҡ„пјҹ
  • и®ҫи®ЎдёҺе®һи·өдёҠзҡ„дёҖдәӣе»әи®®

VS Code жҸ’д»¶пјҡиҮӘе®ҡд№үи§ҶеӣҫвҖ”вҖ”TreeView дёҺ Webview е…ёеһӢеңәжҷҜ

  • TreeViewпјҡз»“жһ„еҢ–ж•°жҚ®зҡ„гҖҢдҫ§иҫ№ж Ҹж ‘гҖҚ
    • 1. TreeView жҳҜд»Җд№ҲпјҢйҖӮеҗҲз”ЁжқҘеҒҡд»Җд№Ҳпјҹ
    • 2. еңЁ package.json дёӯеЈ°жҳҺи§Ҷеӣҫпјҡcontributes.views
    • 3. еңЁд»Јз Ғдёӯе®һзҺ° TreeDataProvider
    • 4. еңЁ activate дёӯжіЁеҶҢ TreeView
  • Webviewпјҡе®Ңе…ЁиҮӘе®ҡд№үзҡ„ HTML и§Ҷеӣҫ
    • 1. Webview жҳҜд»Җд№ҲпјҢйҖӮеҗҲз”ЁжқҘеҒҡд»Җд№Ҳпјҹ
    • 2. еҲӣе»ә Webview Panel зҡ„еҹәжң¬жөҒзЁӢ

  • TreeView vs WebviewпјҡеҰӮдҪ•йҖүжӢ©пјҹ
  • еёёи§Ғй—®йўҳдёҺе®һи·өе»әи®®

Theia жү©еұ•пјҡиҮӘе®ҡд№үи§ҶеӣҫвҖ”вҖ”з”Ё Lumino дёҺ ViewContribution жҢӮдёҖдёӘдҫ§иҫ№ж Ҹйқўжқҝ

    • 2. жүӢеҶҷ bind зҡ„ж ёеҝғжҖқи·Ҝ
  • е…ёеһӢеңәжҷҜйўҳпјҡthisгҖҒдәӢд»¶гҖҒеӣһи°ғйҮҢзҡ„з”Ёжі•
    • 1. гҖҢеҰӮдҪ•еңЁеӣһи°ғйҮҢдҝқиҜҒ this жҢҮеҗ‘жҹҗдёӘеҜ№иұЎпјҹгҖҚ
    • 2. гҖҢеҰӮдҪ•еҖҹз”ЁеҲ«зҡ„еҜ№иұЎзҡ„ж–№жі•пјҹгҖҚ
  • е°Ҹз»“пјҡдёҖйқўйҮҢеҰӮдҪ•и®Іжё… call / apply / bindпјҹ

еүҚз«ҜдёҖйқўпјҡд»Һиҫ“е…Ҙ URL еҲ°йЎөйқўжёІжҹ“пјҢйқўиҜ•е®ҳеҲ°еә•жғіеҗ¬е•Ҙ

  • дёҖйқўжңҹжңӣзҡ„вҖңзІ’еәҰвҖқе’Ңз»“жһ„
  • йҳ¶ж®өдёҖпјҡзҪ‘з»ңиҜ·жұӮзҡ„й«ҳеұӮжөҒзЁӢ
  • йҳ¶ж®өдәҢпјҡHTML и§ЈжһҗдёҺиө„жәҗеҸ‘зҺ°
  • йҳ¶ж®өдёүпјҡд»Һ DOM/CSSOM еҲ° Render TreeгҖҒLayout е’Ң Paint
  • йҳ¶ж®өеӣӣпјҡJavaScript жү§иЎҢдёҺеҜ№жёІжҹ“зҡ„еҪұе“Қ
  • еёёи§ҒйқўиҜ•йўҳдёҺеҸӮиҖғзӯ”жЎҲ
    • йўҳ 1пјҡз®Җиҝ°д»Һиҫ“е…Ҙ URL еҲ°йЎөйқўжёІжҹ“зҡ„иҝҮзЁӢ
    • йўҳ 2пјҡдёәд»Җд№Ҳ CSS дјҡйҳ»еЎһжёІжҹ“пјҹдёәд»Җд№Ҳ JS дјҡйҳ»еЎһи§Јжһҗпјҹ
    • йўҳ 3пјҡйҮҚжҺ’е’ҢйҮҚз»ҳзҡ„еҢәеҲ«пјҹжҖҺж ·еҮҸе°‘е®ғд»¬пјҹ

еүҚз«ҜдёҖйқўпјҡHTTP зј“еӯҳгҖҒжң¬ең°еӯҳеӮЁдёҺи·ЁеҹҹеҹәзЎҖ

  • HTTP зј“еӯҳпјҡејәзј“еӯҳдёҺеҚҸе•Ҷзј“еӯҳ
  • жң¬ең°еӯҳеӮЁпјҡCookieгҖҒlocalStorageгҖҒsessionStorageгҖҒIndexedDB
  • еҗҢжәҗзӯ–з•ҘдёҺи·ЁеҹҹеҹәзЎҖ
    • CORSпјҲи·Ёжәҗиө„жәҗе…ұдә«пјү
    • е…¶е®ғи·ЁеҹҹжүӢж®өпјҲдәҶи§ЈеҚіеҸҜпјү
  • еёёи§ҒйқўиҜ•йўҳдёҺеҸӮиҖғзӯ”жЎҲ
    • йўҳ 1пјҡејәзј“еӯҳе’ҢеҚҸе•Ҷзј“еӯҳжңүд»Җд№ҲеҢәеҲ«пјҹеҲҶеҲ«еҰӮдҪ•жҺ§еҲ¶пјҹ
    • йўҳ 2пјҡCookie е’Ң localStorage жңүд»Җд№ҲеҢәеҲ«пјҹжҖҺд№ҲйҖүз”Ёпјҹ
    • йўҳ 3пјҡз®ҖеҚ•иҜҙиҜҙд»Җд№ҲжҳҜи·ЁеҹҹпјҹеҰӮдҪ•з”Ё CORS и§ЈеҶіпјҹ

еүҚз«ҜдёҖйқўпјҡеүҚз«Ҝе®үе…ЁеҹәзЎҖпјҲXSS дёҺ CSRFпјү

  • XSSпјҲи·Ёз«ҷи„ҡжң¬ж”»еҮ»пјүпјҡжҠҠжҒ¶ж„Ҹи„ҡжң¬вҖңеЎһиҝӣдҪ зҡ„йЎөйқўвҖқ
    • XSS зҡ„еёёи§ҒйҳІеҫЎжүӢж®ө
  • CSRFпјҲи·Ёз«ҷиҜ·жұӮдјӘйҖ пјүпјҡеҲ©з”ЁвҖңжөҸи§ҲеҷЁиҮӘеҠЁеёҰ CookieвҖқ
    • CSRF зҡ„еёёи§ҒйҳІеҫЎжүӢж®ө
  • еёёи§ҒйқўиҜ•йўҳдёҺеҸӮиҖғзӯ”жЎҲ
    • йўҳ 1пјҡд»Җд№ҲжҳҜ XSSпјҹеҰӮдҪ•йҳІеҫЎпјҹ
    • йўҳ 2пјҡд»Җд№ҲжҳҜ CSRFпјҹеҰӮдҪ•йҳІеҫЎпјҹ
    • йўҳ 3пјҡеүҚз«ҜеңЁж—ҘеёёејҖеҸ‘дёӯиҝҳйңҖиҰҒе…іжіЁе“Әдәӣе®үе…ЁзӮ№пјҹ

еүҚз«ҜдёҖйқўпјҡзӣ’жЁЎеһӢгҖҒBFC дёҺз»Ҹе…ёеёғеұҖйўҳ

  • зӣ’жЁЎеһӢпјҡеҶ…е®№гҖҒеҶ…иҫ№и·қгҖҒиҫ№жЎҶе’ҢеӨ–иҫ№и·қ
  • BFCпјҲеқ—зә§ж јејҸеҢ–дёҠдёӢж–ҮпјүпјҡдёҖдёӘвҖңзӢ¬з«Ӣзҡ„е°ҸеёғеұҖдё–з•ҢвҖқ
  • з»Ҹе…ёеёғеұҖйўҳпјҡж°ҙе№іеұ…дёӯгҖҒеһӮзӣҙеұ…дёӯгҖҒдёӨж Ҹ/дёүж ҸеёғеұҖ
  • еёёи§ҒйқўиҜ•йўҳдёҺеҸӮиҖғзӯ”жЎҲ
    • йўҳ 1пјҡж ҮеҮҶзӣ’жЁЎеһӢе’Ң IE зӣ’жЁЎеһӢжңүд»Җд№ҲеҢәеҲ«пјҹеҰӮдҪ•еҲҮжҚўпјҹ
    • йўҳ 2пјҡд»Җд№ҲжҳҜ BFCпјҹе®ғиғҪи§ЈеҶіе“Әдәӣй—®йўҳпјҹ
    • йўҳ 3пјҡз”Ё Flex е®һзҺ°дёҖдёӘе·ҰеҸіеёғеұҖпјҢе·Ұиҫ№еӣәе®ҡ 200pxпјҢеҸіиҫ№иҮӘйҖӮеә”

еүҚз«ҜдёҖйқўпјҡCSS дјҳе…Ҳзә§гҖҒйҖүжӢ©еҷЁдёҺж ·ејҸеҶІзӘҒжҺ’жҹҘ

  • CSS дјҳе…Ҳзә§зҡ„еҹәжң¬и§„еҲҷ
  • еёёи§ҒйҖүжӢ©еҷЁдёҺжҳ“й”ҷзӮ№
  • ж ·ејҸеҶІзӘҒжҺ’жҹҘзҡ„еёёз”ЁжҖқи·Ҝ
  • еёёи§ҒйқўиҜ•йўҳдёҺеҸӮиҖғзӯ”жЎҲ
    • йўҳ 1пјҡдёӢйқўе“ӘжқЎи§„еҲҷз”ҹж•Ҳпјҹдёәд»Җд№Ҳпјҹ
    • йўҳ 2пјҡеҰӮдҪ•ејәеҲ¶жҹҗжқЎж ·ејҸиҰҶзӣ–е…¶е®ғи§„еҲҷпјҹ!important жңүд»Җд№Ҳй—®йўҳпјҹ
    • йўҳ 3пјҡеҰӮдҪ•жҺ’жҹҘдёҖж®өж ·ејҸвҖңдјјд№ҺжІЎз”ҹж•ҲвҖқзҡ„й—®йўҳпјҹ

еүҚз«ҜдёҖйқўпјҡCSS еҠЁж•ҲвҖ”вҖ”transition дёҺ animation еёёи§ҒиҖғзӮ№

  • transition е’Ң animation жң¬иҙЁдёҠжңүд»Җд№ҲеҢәеҲ«пјҹ
  • transitionпјҡж ёеҝғеұһжҖ§дёҺеёёи§Ғиё©еқ‘
    • 1. transition зҡ„е…ій”®еұһжҖ§
    • 2. дёәд»Җд№Ҳ transition жңүж—¶еҖҷгҖҢдёҚз”ҹж•ҲгҖҚпјҹ
    • 3. жҖ§иғҪзӣёе…ізҡ„жіЁж„ҸзӮ№
  • animationпјҡе…ій”®её§ + жҺ§еҲ¶еҸӮж•°
    • 1. @keyframesпјҡе®ҡд№үгҖҢеү§жң¬гҖҚ
    • 2. animation зҡ„еёёз”ЁеұһжҖ§
    • 3. animation еёёи§Ғеқ‘
  • д»Җд№Ҳж—¶еҖҷз”Ё transitionпјҢд»Җд№Ҳж—¶еҖҷз”Ё animationпјҹ
  • йқўиҜ•йҮҢеёёи§Ғзҡ„е»¶дјёй—®йўҳ
    • 1. гҖҢtransition е’Ң animation зҡ„жң¬иҙЁеҢәеҲ«жҳҜд»Җд№ҲпјҹгҖҚ
    • 2. гҖҢдёәд»Җд№ҲжҲ‘зҡ„еҠЁз”»зңӢиө·жқҘеҫҲеҚЎпјҹжңүе“ӘдәӣдјҳеҢ–жүӢж®өпјҹгҖҚ
    • 3. гҖҢеҰӮдҪ•е…јйЎҫеҠЁж•Ҳе’ҢеҸҜи®ҝй—®жҖ§пјҹпјҲprefers-reduced-motionпјүгҖҚ
  • е°Ҹз»“пјҡеҰӮдҪ•еңЁдёҖйқўйҮҢи®Іжё…жҘҡ CSS еҠЁж•Ҳпјҹ

еүҚз«ҜдёҖйқўпјҡDOM дәӢд»¶жөҒгҖҒдәӢд»¶е§”жүҳдёҺеёёи§Ғеқ‘

  • дәӢд»¶жөҒзҡ„дёүдёӘйҳ¶ж®өпјҡжҚ•иҺ·гҖҒзӣ®ж ҮгҖҒеҶ’жіЎ
  • дәӢд»¶е§”жүҳпјҡжҠҠзӣ‘еҗ¬еҷЁжҢӮеңЁзҲ¶е…ғзҙ дёҠ
  • еёёи§Ғеқ‘пјҡйҳ»жӯўеҶ’жіЎгҖҒйҳ»жӯўй»ҳи®ӨиЎҢдёәдёҺ this/target еҢәеҲҶ
  • еёёи§ҒйқўиҜ•йўҳдёҺеҸӮиҖғзӯ”жЎҲ
    • йўҳ 1пјҡиҜҙдёҖдёӢ DOM дәӢд»¶жөҒзҡ„дёүдёӘйҳ¶ж®өпјҢд»ҘеҸҠеҰӮдҪ•еңЁжҚ•иҺ·йҳ¶ж®өз»‘е®ҡдәӢд»¶
    • йўҳ 2пјҡз®Җиҝ°дәӢд»¶е§”жүҳзҡ„еҺҹзҗҶе’ҢдјҳзӮ№
    • йўҳ 3пјҡиҝҷж®өд»Јз Ғдјҡиҫ“еҮәд»Җд№ҲпјҹеҰӮдҪ•йҳ»жӯў inner зҡ„зӮ№еҮ»з»§з»ӯеҶ’жіЎпјҹ
  • е°Ҹз»“пјҡдәӢд»¶зӣёе…ійўҳзӣ®жӣҙзңӢйҮҚвҖңиҝҮзЁӢи®Іи§ЈвҖқиҖҢдёҚеҸӘжҳҜз»“жһң

еүҚз«ҜдёҖйқўпјҡйҳІжҠ–гҖҒиҠӮжөҒдёҺй«ҳйў‘дәӢд»¶дјҳеҢ–

  • й«ҳйў‘дәӢд»¶зҡ„й—®йўҳпјҡscrollгҖҒresizeгҖҒinputгҖҒmousemove
  • йҳІжҠ–пјҲDebounceпјүпјҡеҸӘеңЁвҖңжңҖеҗҺдёҖж¬ЎвҖқи§ҰеҸ‘еҗҺжү§иЎҢ
  • иҠӮжөҒпјҲThrottleпјүпјҡеңЁдёҖж®өж—¶й—ҙеҶ…жңҖеӨҡжү§иЎҢдёҖж¬Ў
  • еёёи§ҒжүӢеҶҷпјҡз®ҖеҚ•зүҲжң¬зҡ„йҳІжҠ–е’ҢиҠӮжөҒ
    • йҳІжҠ–еҮҪж•°пјҲз®ҖеҢ–зүҲпјү
    • иҠӮжөҒеҮҪж•°пјҲж—¶й—ҙжҲізүҲжң¬пјү
  • еёёи§ҒйқўиҜ•йўҳдёҺеҸӮиҖғзӯ”жЎҲ
    • йўҳ 1пјҡд»Җд№ҲжҳҜйҳІжҠ–пјҹд»Җд№ҲжҳҜиҠӮжөҒпјҹеҗ„иҮӘйҖӮз”ЁдәҺе“ӘдәӣеңәжҷҜпјҹ
    • йўҳ 2пјҡеҰӮдҪ•з”Ёд»Јз Ғе®һзҺ°дёҖдёӘйҳІжҠ–еҮҪж•°пјҹ
    • йўҳ 3пјҡеҰӮдҪ•з”Ёд»Јз Ғе®һзҺ°дёҖдёӘиҠӮжөҒеҮҪж•°пјҹж—¶й—ҙжҲіе’Ңе®ҡж—¶еҷЁзүҲжң¬жңүд»Җд№ҲеҢәеҲ«пјҹ

еүҚз«ҜдёҖйқўпјҡе№¶еҸ‘жҺ§еҲ¶гҖҒйҮҚиҜ•дёҺиҪ®иҜўйўҳзӣ®зІҫи®І

  • е№¶еҸ‘йҷҗеҲ¶пјҡеҗҢдёҖж—¶й—ҙжңҖеӨҡи·‘ N дёӘиҜ·жұӮ
  • иҜ·жұӮйҮҚиҜ•пјҡеӨұиҙҘеҗҺжҢүдёҖе®ҡзӯ–з•ҘеҶҚиҜ•еҮ ж¬Ў
  • иҪ®иҜўпјҡе®ҡжңҹиҜ·жұӮзӣҙеҲ°ж»Ўи¶іжқЎд»¶жҲ–и¶…ж—¶
  • еёёи§ҒйқўиҜ•йўҳдёҺеҸӮиҖғзӯ”жЎҲ
    • йўҳ 1пјҡе®һзҺ°дёҖдёӘеёҰе№¶еҸ‘йҷҗеҲ¶зҡ„иҜ·жұӮи°ғеәҰеҷЁ
    • йўҳ 2пјҡе®һзҺ°дёҖдёӘеёҰйҮҚиҜ•зҡ„иҜ·жұӮеҮҪж•°
    • йўҳ 3пјҡе®һзҺ°дёҖдёӘз®ҖеҚ•зҡ„иҪ®иҜўеҮҪж•°

еүҚз«ҜдёҖйқўпјҡеҠ иҪҪдёҺиҝҗиЎҢж—¶жҖ§иғҪдјҳеҢ–ж ёеҝғжҖқи·Ҝ

  • еҠ иҪҪйҳ¶ж®өпјҡи®©йЎөйқўвҖңе°Ҫеҝ«еҸҜз”ЁвҖқ
  • иҝҗиЎҢж—¶йҳ¶ж®өпјҡи®©дәӨдә’вҖңйЎәж»‘дёҚеҚЎйЎҝвҖқ
  • еёёи§ҒйқўиҜ•йўҳдёҺеҸӮиҖғзӯ”жЎҲ
    • йўҳ 1пјҡдҪ еңЁйЎ№зӣ®йҮҢеҒҡиҝҮе“ӘдәӣйҰ–еұҸдјҳеҢ–пјҹеӨ§иҮҙжҖқи·ҜжҳҜжҖҺж ·зҡ„пјҹ
    • йўҳ 2пјҡй•ҝеҲ—иЎЁжёІжҹ“ж—¶дјҡеҚЎпјҢйҖҡеёёжҖҺд№ҲдјҳеҢ–пјҹ
    • йўҳ 3пјҡеҰӮдҪ•жҺ’жҹҘдёҖдёӘйЎөйқўиҝҗиЎҢж—¶еҚЎйЎҝзҡ„й—®йўҳпјҹ

еүҚз«ҜдёҖйқўпјҡжү“еҢ…гҖҒжЁЎеқ—еҢ–дёҺ Tree-Shaking иҖғзӮ№

  • жЁЎеқ—еҢ–пјҡESM vs CommonJS
  • жү“еҢ…е·Ҙе…·еңЁеҒҡд»Җд№ҲпјҹпјҲд»Ҙ Webpack/Vite дёәдҫӢпјү
  • Tree-Shakingпјҡж¶ҲйҷӨжӯ»д»Јз Ғзҡ„еүҚжҸҗе’ҢеұҖйҷҗ
  • еёёи§ҒйқўиҜ•йўҳдёҺеҸӮиҖғзӯ”жЎҲ
    • йўҳ 1пјҡз®ҖеҚ•иҜҙдёӢ ESM е’Ң CommonJS зҡ„еҢәеҲ«пјҹдёәд»Җд№Ҳ ESM жӣҙйҖӮеҗҲеҒҡ Tree-Shakingпјҹ
    • йўҳ 2пјҡжү“еҢ…е·Ҙе…·зҡ„дё»иҰҒе·ҘдҪңжҳҜд»Җд№ҲпјҹејҖеҸ‘зҺҜеўғе’Ңз”ҹдә§зҺҜеўғжңүд»Җд№Ҳе·®ејӮпјҹ
    • йўҳ 3пјҡTree-Shaking дёҖе®ҡиғҪз§»йҷӨжүҖжңүжңӘдҪҝз”Ёд»Јз Ғеҗ—пјҹжңүжІЎжңүеӨұиҙҘзҡ„жғ…еҶөпјҹ

43      ts.isPropertyAccessExpression(node.expression) &&

36function findNodeAtPosition(
37  sourceFile: ts.SourceFile,
38  offset: number
